# HALO PSA API Gateway MCP Server

A Model Context Protocol (MCP) server that provides seamless integration with HALO PSA API endpoints. This server allows Claude Desktop to interact with HALO PSA, search for API endpoints, execute API calls, and manage frequently used queries through Fast Memory.

## Features

- 🔍 **Smart API Endpoint Search**: Search through 1,391 HALO PSA API endpoints using keywords or natural language
- 🚀 **Direct API Execution**: Execute API calls with automatic OAuth2 authentication
- 💾 **Fast Memory**: Save and quickly recall frequently used API queries
- 📚 **Comprehensive Documentation**: Get detailed information about endpoints, parameters, and responses
- 🔐 **Secure Authentication**: OAuth2 client credentials flow with automatic token refresh
- 🏢 **Multi-tenant Support**: Support for HALO PSA's multi-tenant architecture

See full documentation at: https://github.com/jasondsmith72/HALO-PSA-API-Gateway-MCP
